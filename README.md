# Excel Data Cleaning Project

## Overview
This project demonstrates the process of cleaning and analyzing a customer dataset using Microsoft Excel.

## Tasks Performed
- Removed duplicate records.
- Handled missing values.
- Used Excel functions to prepare and organize the data.
- Created PivotTables to summarize customer information.
- Built charts and an interactive dashboard to visualize key insights.

## Tools Used
- Microsoft Excel
- PivotTables
- Charts
- Dashboard
- Excel Functions

- Data Cleaning
- Data Analysis
- Data Visualization
